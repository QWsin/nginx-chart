# nginx-chart
a sample helm chart project with nginx

## Usage

[Helm](https://helm.sh) must be installed to use the charts.  Please refer to
Helm's [documentation](https://helm.sh/docs) to get started.

clone the repo and then install like this:

To install the nginx-chart chart:

    helm install my-nginx-chart <folder path to nginx-chart>

To uninstall the chart:

    helm uninstall my-nginx-chart
